# Dundalk Institute of Technology — Institutional Guidance on Learning Outcomes

DkIT integrates LO guidance into its Quality Assurance framework and Recognition of Prior Learning (RPL) documentation, deferring to QQI "Assessment and Standards" protocols.

## Definition (verbatim)

> "Clear statements of transferable knowledge, skills and attributes which an applicant can be expected to have gained on successful completion of a programme or element of a programme of study e.g. module. The learning outcomes are listed in the module descriptor."

## Standard Stem

> "On successful completion of this module the learner will be able to:"

Followed by action verbs: Identify, Apply, Demonstrate, Evaluate (from observed module descriptors).

## Module Constraints

- Standard module size: 5 ECTS credits
- LOs must be robustly assessable: "Where exemptions are being sought for an award level module then the assessment process should assign a mark for the module arising out of a robust assessment of the learning achieved as defined by the assessment criteria of the module."

## Verb Requirements

Bloom's taxonomy verbs used throughout internal descriptors. Active verb required. No explicit prohibited term list found — QQI/Cedefop standards apply by default.

## Notes for AI Evaluator

- Standard Irish HE conventions apply (active verb, student-centred stem, NFQ level alignment)
- Robust assessability is an explicit requirement
- 5 ECTS standard module — apply standard scope calibration
- DkIT defers primarily to QQI; apply QQI/Cedefop/Kennedy 2006 as evaluative baseline
