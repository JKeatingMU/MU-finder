# Munster Technological University — Institutional Guidance on Learning Outcomes

MTU (Extended Campus) provides a specific structural formula for LO writing, notably using **SOLO taxonomy** (Structure of Observed Learning Outcomes) as an alternative or supplement to Bloom's. Source: *Writing Learning Outcomes* (MTU Extended Campus) and the ECAAPT (Embedding Content Across Academic Programmes Toolkit) guidance.

## LO Formula (verbatim)

MTU mandates a four-part syntax:

> "**Verb** – defines the level of understanding; shows the depth of competences (SOLO).
>
> **Object** – an aim the activities are directed (write an essay; build a robot; manage a project...)
>
> **Adverb** – shows the depth of competences.
>
> **Context** – shows the extent of the competences."

This is Verb + Object + Adverb + Context — more structured than the standard Kennedy formula and explicitly SOLO-referenced for verb selection.

## Prohibited Phrasing (verbatim)

MTU actively prohibits overly restrictive phrasing — an unusual requirement not found in other Irish HEI guidance:

> "There should not be any unnecessary limitations in learning outcomes about specific theories, approach to the task or for example used software. These limitations are acceptable only if the competences are so specific, that there is only one possible way to do things. The main idea is, that gained/proved competences count, not the exact way of achieving the correct final result."

This is the over-specification problem: an LO that specifies a particular tool, theory, or method where alternatives would demonstrate the same competence is considered poorly written.

## Module Structure

- Standard modules: 5 ECTS credits
- LOs mapped via ECAAPT to ensure progressive levelling across the programme
- Outcomes should not be "overcrowded" — implied quantity constraint but no explicit count given

## Notes for AI Evaluator

- SOLO taxonomy is the primary verb-selection framework (not Bloom's) — verbs signal learning depth across unistructural → multistructural → relational → extended abstract levels
- Four-part formula: Verb + Object + Adverb + Context — flag LOs missing object or context
- Over-specification is a distinct quality problem at MTU: LOs naming specific tools/theories unnecessarily should be flagged
- Under-specification (vague verb) remains the primary problem; over-specification is secondary
- NFQ level alignment expected but not elaborated in found guidance
- MTU is a merged institution (CIT + IT Tralee) — apply consistent QQI/NFQ baseline across campuses
