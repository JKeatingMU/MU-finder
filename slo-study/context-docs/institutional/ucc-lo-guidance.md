## Definition (verbatim, Summary Guidelines)

> "Learning outcomes are statements of what a student should know, understand and be able to demonstrate after completion of a process of learning."

Three domains: cognitive (knowledge-based), affective (attitudes/values), psychomotor (practical skills). Most module LOs are cognitive.

## Rules for Writing LOs (verbatim, Summary Guidelines p.2)

> 1. **Always use action verbs.** Think about completing the sentence *"At the end of this module students should be able to:"*
> 2. **Keep the sentence short.** More than one action verb can be used in the same sentence.
> 3. **Try to ensure that module Learning Outcomes range across all levels of Bloom's Taxonomy** in each year of the programme.
> 4. **Programme Learning Outcomes should map on to the appropriate level of the National Qualifications Framework.**

Key evaluative question: *"What must the students be able to DO in order to show that they have achieved the Learning Outcome?"*

## Prohibited Terms

LOs must not use aim/objective language (teacher-centred). Prohibited phrasings (explicit):
- "To give students an understanding of..."
- "To make students familiar with..."
- "To ensure that students know..."
- "To enable students to experience..."

Kennedy (2006) lists the most common mistake as using vague terms: *know, understand, learn, be familiar with, be exposed to, be acquainted with, be aware of*.

## Quantity Requirements (policy-mandated)

- **Module LOs: 3–9** (Module Policy for CIM Modules, confirmed)
- **Programme LOs: 5–10**

Module Policy verbatim: "All modules require a minimum of three and a maximum of nine learning outcomes, reflected as a numbered list."

## Bloom's Taxonomy Verb Table (Summary Guidelines, original taxonomy)

| Level | Sample verbs |
|---|---|
| Knowledge | Define, Describe, Identify, List, Name, Recall, Recognise, State |
| Comprehension | Classify, Explain, Interpret, Predict, Summarise, Translate |
| Application | Apply, Calculate, Demonstrate, Solve, Use |
| Analysis | Analyse, Compare, Contrast, Differentiate, Examine, Investigate |
| Synthesis | Compose, Construct, Create, Design, Devise, Formulate, Plan |
| Evaluation | Appraise, Assess, Evaluate, Judge, Justify, Recommend |

Affective domain: Appreciate, Demonstrate (a belief in), Discuss, Embrace, Value, etc.
Psychomotor domain: Grasp, Handle, Manipulate, Operate, Perform, etc.

## QQI/NFQ Relationship

UCC requires programme LOs to map to the NFQ Grid of Level Indicators. However, UCC adds substantial supplementary guidance beyond QQI's minimum:

| Requirement | QQI/NFQ | UCC |
|---|---|---|
| NFQ level alignment | Yes | Yes (required) |
| Specific verb requirement | No | Yes (action verbs, Bloom's) |
| Prohibited terms | No | Yes (explicit list) |
| LO count per module | No | Yes (3–9, policy-mandated) |
| LO count per programme | No | Yes (5–10) |
| Constructive alignment with assessment | No | Yes (institutional expectation) |
| Prescribed stem sentence | No | Yes |

## Notes for AI Evaluator

- "know", "understand", "be familiar with", "be exposed to", "be acquainted with", "be aware of" are explicitly prohibited
- 3–9 LOs per module is policy (flag outliers)
- Bloom's verb range across levels is a stated institutional expectation
- NFQ level alignment required at programme level
- LOs must be evidenced by module assessment profile (constructive alignment)
- Stem: "At the end of this module students should be able to:"
